## 5-paragraph

Universal Manifest v0.2 was published on 2026-04-30. The durable versioned specification is https://universalmanifest.net/spec/v0.2/, and the canonical publication record is the changelog entry at https://universalmanifest.net/publishing/changelog/#2026-04-30--v02-published.

This release moves the v0.2 signature profile into a published posture for implementers. It defines the Ed25519 and JCS-based integrity path, keeps the versioned artifact URLs stable, and gives external consumers a concrete target for validation.

Existing v0.1 consumers and issuers should start with the migration guide at https://universalmanifest.net/guides/migration-v01-v02/. The guide is the upgrade path for teams deciding when to parse v0.2, when to verify signatures, and how to operate during a mixed-version period.

Implementation claims should be checked against the v0.2 conformance material and pinned artifacts referenced from the changelog. The changelog remains the canonical announcement source if shorter home-page, email, or social copy ever differs from the detailed release record.

Questions, implementation reports, and proposed spec changes should go through https://github.com/grigb/universal-manifest/issues. Security issues should not be filed publicly; use the security contact published by the project instead.
