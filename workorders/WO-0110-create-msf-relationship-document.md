## Objective

Document UM's current and future relationship with the Metaverse Standards Forum (MSF), given the MUM (Metaverse Universal Manifest) lineage.

## Why this work matters

The completeness audit found that the MSF is referenced exclusively in the lineage context ("UM is descended from MUM, created in the MSF repo") but there is no documentation about UM's current or future relationship with the MSF as an organization. Given that UM originated in an MSF working group, the project should explicitly state whether there are plans to contribute UM back to MSF working groups or whether the lineage is purely historical.

## Scope

### Option 1: Standalone document

Create **`docs/MSF-RELATIONSHIP.md`** documenting:
1. MUM lineage — how UM originated in the MSF
2. Current relationship — what UM's current status is relative to the MSF
3. Future plans — whether/when UM might be contributed back to MSF

### Option 2: Section within Standards Positioning document

Add an MSF section to the Standards Positioning document (WO-0093) rather than creating a standalone document.

### Source material

- `docs/PROJECT-VISION.md` — MUM lineage references
- `docs/DECISIONS.md` — MSF-related decisions
- `integrations/metaverse.md` — Metaverse integration lane with MSF lineage context
