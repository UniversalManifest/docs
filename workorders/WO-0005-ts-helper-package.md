# WO-0005 — TS helper package distribution + usage guidance

**Status:** Proposed  
**Created:** 2026-02-12

## Objective

Decide what to do with the TypeScript helper package:

- keep it internal-only for now (private)
- or publish it (e.g., npm, GitHub Packages) for external adopters

## Scope

In scope:

- Document what the package is and what guarantees it provides
- Decide distribution strategy
- If publishing: define versioning and whether it mirrors spec versions

Out of scope:

- Building a full SDK

## Acceptance criteria

- [ ] Decision recorded in `docs/DECISIONS.md`
- [ ] Documentation added (where to import, what functions do, what they do NOT do)
